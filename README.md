# 223_GradeDoc
Eigenes Projekt 223

In diesem Projekt kann man sich registrieren und einloggen.

Probleme:
Man kommt über die oben angegebenen URLs nicht auf meine Forms sondern
wird direkt auf die login Seite von Spring Security geleitet.

Da man nicht auf die Seite kommt, bei der man seine Daten eingeben kann,
schreibt es auch noch nicht alles in die DB (Tabellen und Felder werden erstellt)

Vorbereitung:
1. Eine MySQL Datebank erstellen mit dem namen gradedb
2. Das script in die DB laden

Ausführung:
1. Projekt starten
2. In dem Browser auf localhost:8080/login oder localhost:8080/register
3. Erforderliche Daten angeben
